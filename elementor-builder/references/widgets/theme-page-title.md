# `theme-page-title` — Titre de la page (Pro)

`x`+GROUPE = groupe de contrôles (sous-clés dans common-controls.md). Suffixes responsive `_tablet`/`_mobile` dispo sur les contrôles responsive.

## onglet content
- section_title : `title`:txt, `link`:url{url,is_external}, `size`(default|small|medium|large|xl|xxl)[=default], `header_size`(h1|h2|h3|h4|h5|h6|div|span|p)[=h1]

## onglet style
- section_title_style : `typography_typography`+GROUPE typo, `text_stroke_text_stroke_type`+GROUPE text_stroke, `align`(start|center|end|justify), `text_shadow_text_shadow_type`, `text_shadow_text_shadow`:text_shadow, `blend_mode`(|multiply|screen|overlay|darken|lighten|color-dodge|saturation|color|difference|…), `title_colors`, `title_color`:color, `title_hover_color`:color, `title_hover_color_transition_duration`:slider{unit,size}
