# `progress` — Barre de progression (core)

`x`+GROUPE = groupe de contrôles (sous-clés dans common-controls.md). Suffixes responsive `_tablet`/`_mobile` dispo sur les contrôles responsive.

## onglet content
- section_progress : `title`:txt[=Ma compétence], `title_tag`(h1|h2|h3|h4|h5|h6|div|span|p)[=span], `title_display`:'yes'[=yes], `progress_type`(|info|success|warning|danger), `percent`:slider{unit,size}, `display_percentage`:'yes'[=show], `inner_text`:txt[=Concepteur web]

## onglet style
