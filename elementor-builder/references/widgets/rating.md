# `rating` — Notation (core)

`x`+GROUPE = groupe de contrôles (sous-clés dans common-controls.md). Suffixes responsive `_tablet`/`_mobile` dispo sur les contrôles responsive.

## onglet content
- section_rating : `rating_scale`:slider{unit,size}, `rating_value`:nb, `rating_icon`:icons{value,library}, `icon_alignment`(start|center|end)

## onglet style
- section_icon_style : `icon_size`:slider{unit,size}, `icon_gap`:slider{unit,size}, `icon_color`:color, `icon_unmarked_color`:color
