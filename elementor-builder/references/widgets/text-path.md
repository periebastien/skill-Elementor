# `text-path` — Chemin du texte (core)

`x`+GROUPE = groupe de contrôles (sous-clés dans common-controls.md). Suffixes responsive `_tablet`/`_mobile` dispo sur les contrôles responsive.

## onglet content
- section_content_text_path : `text`:txt, `path`(wave|arc|circle|line|oval|spiral|custom)[=wave], `custom_path`:media{id,url}, `link`:url{url,is_external}, `align`(left|center|right), `text_path_direction`(|rtl|ltr), `show_path`:'yes'

## onglet style
- section_style_text_path : `text_typography_typography`+GROUPE typo, `text_stroke_text_stroke_type`+GROUPE text_stroke, `size`:slider{unit,size}, `rotation`:slider{unit,size}, `word_spacing`:slider{unit,size}, `start_point`:slider{unit,size}, `text_style`, `text_color_normal`:color, `text_color_hover`:color, `hover_animation`, `hover_transition`:slider{unit,size}
- section_style_path : `path_style`, `path_fill_normal`:color, `stroke_color_normal`:color[=#E8178A], `stroke_width_normal`:slider{unit,size}, `path_fill_hover`:color, `stroke_color_hover`:color, `stroke_width_hover`:slider{unit,size}, `stroke_transition`:slider{unit,size}
