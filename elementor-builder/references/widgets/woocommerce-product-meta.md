# `woocommerce-product-meta` — Métas produit (Pro)

`x`+GROUPE = groupe de contrôles (sous-clés dans common-controls.md). Suffixes responsive `_tablet`/`_mobile` dispo sur les contrôles responsive.

## onglet style
- section_product_meta_style : `text_typography_typography`+GROUPE typo, `link_typography_typography`+GROUPE typo, `view`(table|stacked|inline)[=inline], `space_between`:slider{unit,size}, `divider`:'yes', `divider_style`(solid|double|dotted|dashed)[=solid], `divider_weight`:slider{unit,size}, `divider_width`:slider{unit,size}, `divider_height`:slider{unit,size}, `divider_color`:color[=#ddd], `text_color`:color, `link_color`:color
- section_product_meta_captions : `category_caption_single`:txt, `category_caption_plural`:txt, `tag_caption_single`:txt, `tag_caption_plural`:txt, `sku_caption`:txt, `sku_missing_caption`:txt
