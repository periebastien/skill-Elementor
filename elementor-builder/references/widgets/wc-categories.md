# `wc-categories` — Catégories de produit (Pro)

`x`+GROUPE = groupe de contrôles (sous-clés dans common-controls.md). Suffixes responsive `_tablet`/`_mobile` dispo sur les contrôles responsive.

## onglet content
- section_layout : `columns`:nb[=4], `number`:nb[=4]
- section_filter : `source`(|by_id|by_parent|current_subcategories), `categories`(17|23|16|24), `parent`(0|17|23|16|24)[=0], `hide_empty`:'yes', `orderby`(name|slug|description|count)[=name], `order`(asc|desc)[=desc]

## onglet style
- section_products_style : `image_border_border`+GROUPE bordure, `title_typography_typography`+GROUPE typo, `count_typography_typography`+GROUPE typo, `column_gap`:slider{unit,size}, `row_gap`:slider{unit,size}, `align`(left|center|right), `image_border_radius`:dim{t,r,b,l,unit,isLinked}, `image_spacing`:slider{unit,size}, `title_color`:color, `count_color`:color
